The project

A reinforcement-learning agent that learns to play Snake from scratch using Deep Q-Networks (DQN). The agent observes the grid state and learns a policy that maximises food collected before dying.

How the AI plays

The AI runs the actual trained DQN weights loaded directly from model.pth (Linear_QNet 11→256→3). It observes 11 features — danger straight/right/left, current direction, and food location — and picks the highest-Q action each step.

Race rules

Both boards receive food at the same positions (seeded sequence, seed 42) so neither player has an advantage. A snake that runs into itself freezes — the other keeps going until it dies too.
