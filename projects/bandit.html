Multi-Armed Bandit : Nudge Timing

Watch an epsilon-greedy agent learn the best hour to trigger a push notification — CTR climbs from 2.4% to 7.4%.

Each hour (8 AM–11 PM) is a bandit arm. Click an hour to send a simulated push notification at that time. The epsilon-greedy agent (ε = 0.15) mixes exploration with exploitation to find the arm with the highest click-through rate. Best hour (after enough pulls): ?
Current CTR
2.4%

Highlighted bar = highest estimated CTR so far. The true best arm is 2 PM at 7.4%.

The project

Identifies the optimal time of day to send a push notification to maximise click-through rate. Deployed at JM Financial - raised production CTR from 2.4% to 7.4%.

Algorithm

Epsilon-greedy multi-armed bandit (ε = 0.15): with 15% probability the agent explores a random arm; otherwise it exploits the arm with the highest observed reward. Q-values (estimated CTRs) are updated after every pull.

Convergence

Click "Auto-run 200 pulls" to watch the algorithm converge in seconds - the bar for 2 PM grows taller as the agent discovers it has the highest true CTR (7.4%).
