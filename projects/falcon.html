Falcon 9 Cost Predictor

Adjust payload, orbit, and booster reuse — see estimated launch cost and landing probability update in real time.

$67M
Estimated launch cost · Decision-tree model (87.5% accuracy)

The project

Predicts Falcon 9 launch cost and booster landing probability using payload mass, target orbit, booster reuse history, and launch site. Trained on historical SpaceX mission data.

Model

A decision-tree classifier trained on features extracted from the SpaceX REST API and Wikipedia web scraping. Best model accuracy: 87.5%.

Cost formula

This demo implements the tree's decision logic transparently: base cost by orbit type + payload surcharge + reuse discount + launch-site modifier. The breakdown is shown live below the result.
